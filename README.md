# 스프링부트를 활용하여 ToDoList 만들기 프로젝트

프로젝트 목표
- spring boot, JPA, gradle의 기술을 활용하기
- mysql을 gradle에 세팅하여 활용하기
- thymeleaf와 bootstrap 템플릿 적용하여 활용하기
- flyway를 통해 database schema 쉽게 관리하기


개발환경
 -SpringBoot 2.3.1
 -java 1.8
 -MySQL
 -Gradle 3.0
 
 사용방법
 
 1. 우측상단 Clone or download 클릭
 2. Download ZIP 클릭
 3. 압출 풀기
 4. SpringBoot 에서 열기
 5. 실행하기
